# 梅卦爻断

用梅花易数的方法，进行六爻的判定。

日月：六爻中的日月

天：证监会、国家、股票代码、上证指数或者深证指数，经济环境好坏，决定了一切。

地：股票的公司，他们的决策，才是股价涨跌的核心。公司的名字

人：基金、游资、散户。昨天的收盘价格。

# 仅价格
价格要根据实际来。

# 价格+编码+股票代码
编码是证券，天，整个证券市场是国家的资金池和赌场

公司名字是公司自己，地，公司是基础

价格是市场，人，是投资者的来回投资

最好一次性进行多种算法的排摸


# 流程
1. 根据价格，取卦
2. 根据价格+编码，取卦
3. 根据股票名称取卦
